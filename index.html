Command Git

  • git init: Create an empty Git repository or reinitialize an existing one
  • git add .: Add file contents to the index
  • git status: Show the working tree status
  • git commit -m [judul commit]: Record changes to the repository
  • git log: Show commit logs
  • git log --all --decorate --oneline --graph: Oneline log
  • git checkout: Switch branches or restore working tree files
  • git merge: Join two or more development histories together
  • git branch --merged: Only list branches whose tips are reachable from the specified commit (HEAD if not specified)
  • git branch -d [nama branch]: Delete a branch. The branch must be fully merged in its upstream branch, or in HEAD
  • git branch -D [nama branch]: Shortcut for --delete --force
  • git remote: Manage set of tracked repositories
  • git push: Update remote refs along with associated objects
